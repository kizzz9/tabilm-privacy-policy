Tabilm におけるプライバシーとデータの取り扱いについて

最終更新日 2026.07.08

Summary

要約

Tabilm のプライバシー方針を一言で

旅の記録はあなたの端末と iCloud の中だけで完結します。開発者のサーバーはありません。アクセス解析ツールも広告 SDK も一切組み込んでおらず、開発者があなたの旅や動画を見ることはできません。

Data

取り扱う情報と保存先

何を、どこに、なぜ保存するか

しおり・予定・持ち物

旅のタイトル / 日程 / メモ / しおり予定 / 持ち物リスト。あなたの iCloud（CloudKit）と端末に保存し、端末間で同期します。

撮影クリップ・サムネ

撮影した動画とサムネイル（撮影地点の座標を含むことがあります）は端末ローカルのみ。共有中の旅では軽量化したプロキシ動画とサムネイルのみを iCloud の共有ゾーンへアップロードして同行者と共有しますが、元の高画質素材は撮影した端末にのみ残ります。

取り込んだ写真

写真ライブラリから取り込む旅の写真は端末ローカルのみ。あなたがトグルを ON にした時だけ読み取ります。

書き出した Vlog

「カメラロールへ保存」を選んだ時のみ写真ライブラリへ追加します。

無料書き出し枠の使用状況

初回 1 本無料の使用フラグを端末とあなたの iCloud キーバリューストレージに保存（再インストール時の復元のため）。

App 内課金

購入状態は Apple の StoreKit が管理。決済情報は Apple が処理し、本アプリは取得しません。

アプリ内フィードバック

あなたが任意で送信ボタンを押した時だけ、ご意見テキスト・カテゴリ・アプリ／iOS バージョン・機種名・言語設定を iCloud の公開データベースへ送信します（開発者が閲覧）。氏名・メール・Apple ID など個人を特定する情報は含みません。

Permissions

端末の機能・権限

いずれも拒否したまま他の機能を使えます

  • カメラ / マイク — 撮影中のみ使用します。
  • 位置情報 — 使用中のみ。場所検索（Apple Maps）、しおりの天気表示（Apple Weather に予定地点の座標を送信）、撮影クリップへの位置記録に使います。
  • 写真ライブラリ — 旅の写真の取り込み（読み取り）と、完成した Vlog の保存（追加のみ）。
  • 通知 — 撮影リマインダー等のローカル通知のみ。プッシュ配信サーバーはありません。

Third party

第三者サービス

利用するのは Apple のサービスのみ

  • Apple iCloud / CloudKit — 旅データの保存・同期・共有
  • Apple Weather (WeatherKit) — しおりの天気表示のため予定地点の座標を Apple に送信
  • Apple Maps (MapKit) — 場所検索のため検索文字列を Apple に送信
  • App Store (StoreKit) — App 内課金

いずれも Apple のサービスで、Apple のプライバシーポリシーが別途適用されます。上記以外に外部へデータを送信することはなく、これらの情報を第三者に販売・提供することもありません。

Sharing

同行者と共有したとき

iCloud で旅を共有すると、招待を承諾した相手だけがその旅のしおり・持ち物を閲覧・編集できます。共有を解除すると相手のアクセスは失効します。共有中の旅では、撮影クリップを軽量化したプロキシ動画とサムネイルにして iCloud の共有ゾーンへアップロードし、同行者が合作 Vlog の素材として使えるようにします。各端末は自分が撮ったクリップのプロキシだけをアップロードし、元の高画質素材は撮影した端末にのみ残ります。オーナーはアプリ内の「iCloud の共有素材を削除」からプロキシをまとめて削除できます。

Deletion

データの削除

  • 旅を削除すると、iCloud 上の対応データと端末内の動画・サムネイル・しおりが削除されます。
  • 共有に参加した旅からはいつでも退出できます。
  • アプリを削除すると端末ローカルのデータは消去されます。iCloud 上のデータは iOS の「設定 → Apple ID → iCloud」から管理・削除できます。

Other

子どもの利用 / 改訂

本アプリは 13 歳未満の利用を想定していません。ポリシーを改訂した場合は本ページの最終更新日を変更し、必要に応じてアプリ内で告知します。
